# 📋 Responsive Student Registration Form

## 📖 Overview

This project is a **Responsive Student Registration Form** built using **HTML, CSS, and JavaScript**. Users can enter student details through a form, and the submitted data is displayed dynamically in a table below the form without refreshing the page.

The project demonstrates form handling, DOM manipulation, responsive web design, and JavaScript event handling.

---

## ✨ Features

- Responsive Student Registration Form
- Required field validation
- Gender selection using radio buttons
- Dynamic table row creation
- Prevents page refresh on submission
- Automatically resets the form after submission
- Clean and modern user interface
- Mobile, tablet, and desktop friendly
- Uses Flexbox and Media Queries
- Uses `addEventListener()` for event handling

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

The form includes the following required fields:

## ⚙️ Functionality

### Form Submission
- Validates all required fields
- Prevents page refresh
- Collects user input
- Dynamically adds data to the table
- Resets the form after successful submission

### Dynamic Table
- Displays submitted student records
- Creates a new row for every submission
- Maintains proper alignment and readability

- Desktop-friendly layout
- Tablet-friendly layout
- Mobile-friendly layout
